# フォルム切替の一般化（メガ以外も循環）

## MODIFIED Requirements

### Requirement: メガ進化（フォーム切替）

選択ポケモンの画像をタップするとフォーム切替でき**なければならない（MUST）**。旧 champ-edge の
フォームボタンと同様に、`pokemon_data` にそのポケモンの**複数フォルム**（form 0〜19。通常の別フォルム
1-9 とメガ 10-19 を含む）がある場合、タップで昇順に循環し、最後のフォルムの次は先頭（通常）に戻る。
切り替え時は種族値・タイプ・特性・名前・画像をそのフォルムに差し替え、技・努力値・性格・持ち物・ランク等は
維持する。フォルムが1つだけのポケモンは変化しない。種族値ポップアップや外部サイトリンクは設けない。

#### Scenario: タップでフォルム切替（メガ）

- **WHEN** メガフォームを持つポケモンの画像をタップする
- **THEN** メガフォームに切り替わり、種族値・タイプ・特性を反映してダメージが再計算される

#### Scenario: タップでフォルム切替（通常別フォルム）

- **WHEN** ギルガルド（シールド）の画像をタップする
- **THEN** ギルガルド（ブレード）に切り替わり、種族値・タイプを反映してダメージが再計算される

#### Scenario: 先頭フォルムへ戻る

- **WHEN** 最終フォルムのポケモンの画像をさらにタップする
- **THEN** 先頭（通常）フォルムに戻る
