# 素早さ比較の追加

## ADDED Requirements

### Requirement: 素早さ比較

システムは自分と相手の active ポケモンの素早さを比較できるようにし**なければならない（MUST）**。
ドロワーメニューの「素早さ比較」からポップアップを開き（両者が選択されているときのみ）、旧 champ-edge の
`SpeedComparing` を踏襲して左右に並べて表示する：

- 各側に アイコン・名前・**素早さ実数値（大きく表示）**。速い側を赤・遅い側を青・同速は黒で表示し、
  中央に `>` / `=` / `<` の比較記号を出す。
- 変更できる項目（変更すると即時再計算）：努力値（0-32、0/32 ボタン付き）・性格補正（0.9/1.0/1.1）・
  ランク（−6〜+6、±1 ボタン）・特性倍率（0.5/1.0/1.5/2.0）・道具倍率（0.5/1.0/1.5/2.0）・
  おいかぜ（×2）・まひ（×0.5）。種族値は表示のみ・個体値は 31 固定。
- 実数値は原典と同じ式：`floor((種族値×2+31+努力値×2)×50/100)+5` に性格・特性・道具・おいかぜ・まひ・
  ランク倍率（+n は (n+2)/2、−n は 2/(n+2)）を乗じて小数切り捨て。
- 初期値はポケモンの現在値を自動反映する：種族値S・努力値S・ランクS、素早さ上昇性格
  （おくびょう/せっかち/やんちゃ/ようき）なら 1.1・下降性格（ゆうかん/のんき/れいせい/なまいき）なら 0.9、
  持ち物がこだわりスカーフなら道具 1.5。
- ポップアップを閉じると入力内容は破棄される（ポケモン本体には反映しない）。

#### Scenario: 素早さを比較する

- **WHEN** 両者を選択した状態でメニューの「素早さ比較」を開く
- **THEN** 両者の素早さ実数値が表示され、速い側が赤・遅い側が青で示される

#### Scenario: 項目変更で即時再計算

- **WHEN** 片側のランクを +1 する（またはおいかぜを ON にする）
- **THEN** その側の素早さ実数値と比較記号が即座に更新される

## MODIFIED Requirements

### Requirement: メニュー（隅のボタンから展開）

システムは中央列に操作アイコンとメニューを備え**なければならない（MUST）**。中央列は上から
メニュー（ハンバーガー）／HOME 使用率／類似パーティ検索／対戦記録のアイコン、天候・フィールドの
ドロップダウン、カウンター／タイマーのアイコンを縦に並べる（押しやすいよう間隔を空ける）。
ハンバーガーを押すとドロワーが開き、区切り線で「素早さ比較・重さ比較」「パーティ編集・ボックス編集」
「対戦履歴・対戦分析」「設定」の4グループを表示する。**素早さ比較は実装済み**（「素早さ比較」要件参照）。
残りのアイコン/項目はプレースホルダ（押下で「今後実装予定」）で、段階的に実装してよい。

#### Scenario: メニューを開く

- **WHEN** 中央列のハンバーガーメニューをタップする
- **THEN** 区切り線で分類された各メニュー項目（比較／パーティ／戦績／設定）が表示される
