# 重さ比較・weight 配管・基本情報の重さ表示

## ADDED Requirements

### Requirement: 重さ比較

システムは自分と相手の active ポケモンの重さを比較できるようにし**なければならない（MUST）**。
ドロワーメニューの「重さ比較」からポップアップを開き（両者が選択されているときのみ）、旧 champ-edge の
`WeightComparing` を踏襲して左右に並べて表示する：

- 各側に アイコン・名前・**重さ（kg、大きく表示）**。重い側を赤・軽い側を青・同重量は黒で表示し、
  中央に `>` / `=` / `<` の比較記号を出す。
- その下に「威力」行：各側がヘビーボンバー/ヒートスタンプで相手を攻撃した場合の威力を表示する。
  重量比（相手の重さ ÷ 自分の重さ）が 0.2 未満→120／0.25 未満→100／0.3 未満→80／0.5 未満→60／
  それ以外→40。
- 重さは `pokemon_data.weight` 由来で、ダメージ計算（エンジンの重量比威力）にも同じ値が渡されること。

#### Scenario: 重さを比較する

- **WHEN** 両者を選択した状態でメニューの「重さ比較」を開く
- **THEN** 両者の重さが表示され（重い側が赤）、互いに対するヘビーボンバー系の威力が表示される

#### Scenario: ヘビーボンバーのダメージ計算に重さが反映される

- **WHEN** 重いポケモンで軽いポケモンにヘビーボンバーを選ぶ
- **THEN** 重量比に応じた威力（最大120）でダメージが計算される

## MODIFIED Requirements

### Requirement: 選択ポケモンの基本情報表示

システムは選択中のポケモンの基本情報を表示し**なければならない（MUST）**。表示項目は画像・名前・
タイプ・特性・持ち物・性格・実数値（H/A/B/C/D/S）・努力値・ランク・**重さ（kg）**とする。タイプは小さめの
チップで**ポケモン名の横**に表示し（複数タイプは縦2段、単タイプは1つだけ）、**タップで変更**
できること（バトル中のタイプ変更に対応）。重さは名前の近くに小さく表示する（旧 champ-edge の
基本情報欄の重さ表示に相当）。
実数値は種族値・個体値・努力値（0〜32 スケール）・レベル・性格
から算出すること。テラスタイプとレベルは当面表示しない（テラスは将来再導入候補）。

#### Scenario: 基本情報の表示

- **WHEN** ポケモンを選択する
- **THEN** 画像・名前・タイプ（名前の横）・特性・持ち物・性格・実数値・努力値・ランク・重さが表示される

#### Scenario: タイプ変更

- **WHEN** タイプをタップしてタイプ選択ダイアログから別タイプを選ぶ
- **THEN** そのタイプ枠が変更され、タイプ一致補正・相性を反映してダメージが再計算される
