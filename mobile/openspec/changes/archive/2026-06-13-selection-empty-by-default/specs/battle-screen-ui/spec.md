# 選出は初期空・登録/クリアでは選出しない

## MODIFIED Requirements

### Requirement: パーティと選出（画像・番号表示・最大3体）

各側はパーティ 6 体を**画像**で横並び表示し**なければならない（MUST）**。選出は独立した欄を設けず、
パーティのポケモンに**選んだ順の番号（1〜3、最大 3 体）**を重ねて表示すること。**選出は起動時・
パーティクリア時は空**とし（原典同様、勝手に選出しない）、パーティの
ポケモンをタップすると詳細表示の対象（選択ポケモン）になり、選出が 3 体未満なら選出に追加される。
選出が 3 体のときは通常タップでは選出を変えず、**長押しで選出から解除**する。パーティ編集で**枠に
ポケモンを登録した場合、その枠は選出に追加しない**（既に選出済みなら外す。明示タップで登録する）。
**ポケモンが未設定の枠（空きスロット）は、アイコンを出さず白背景＋枠のみを表示する**（選出番号も出さない）。

#### Scenario: タップで選出と詳細切替

- **WHEN** 選出が 3 体未満の状態でパーティのポケモンをタップする
- **THEN** そのポケモンが選出に番号付きで追加され、選択ポケモンの基本情報・技がそのポケモンに更新される

#### Scenario: 3 体選出済みでのタップ

- **WHEN** 既に 3 体選出済みの状態で未選出のポケモンをタップする
- **THEN** 選出は変化せず、詳細表示のみそのポケモンに切り替わる

#### Scenario: 長押しで選出解除

- **WHEN** 選出中のポケモンをパーティで長押しする
- **THEN** そのポケモンが選出から外れる

#### Scenario: 登録では選出されない

- **WHEN** パーティ編集の枠にポケモンを登録して編集を閉じる
- **THEN** その枠は選出されておらず（番号バッジが付かず）、タップして初めて選出に追加される

#### Scenario: 空きスロットの表示

- **WHEN** パーティに未設定の枠がある
- **THEN** その枠はアイコンを出さず、白背景に枠線のみで表示される（例外やプレースホルダ文字を出さない）
