# 登録/選出時の HOME 自動補完・フォーム名タップ・切替ランククリア

## MODIFIED Requirements

### Requirement: パーティの手動編集

システムは各パーティを手動で編集できるようにし**なければならない（MUST）**。パーティ横の編集ボタンから
編集ダイアログを開き、6 枠のいずれかをタップすると**ポケモンを文字入力検索**（`pokemon_data`）して
そのポケモン（種族値・タイプ・特性を DB から構築）に差し替えられること。編集ダイアログには**クリア
ボタン**があり、6 枠すべてを空にできる。**ポケモンを差し替えた際、および初回選出時（技が空・未補完の実
ポケモン）には、HOME 使用率データから性格・持ち物・特性（候補にある場合）・努力値（合計66の最上位を優先）・
技（上位を最大4つ、DB で解決）を自動補完する**（旧 champ-edge の登録時 HOME 反映に相当）。HOME データが
無いポケモンは空のまま（手動入力）。ポケモン候補は旧 champ-edge と同様に**メガシンカ（name LIKE 'メガ%'、
メガニウム除く）と除外リスト（ヒヒダルマ(ダルマ) 等）を外し、さらに画像があるポケモンのみ**に限定する。
直近の自分・相手パーティは自動保存し、次回起動時にそのまま復元する（旧アプリの「使用中」パーティ相当）。
名前付きの保存・読込 UI は当面設けない（必要になれば後続フェーズで追加）。

#### Scenario: 枠のポケモン差し替え（HOME自動補完）

- **WHEN** パーティ編集ダイアログの枠をタップしてポケモンを検索・選択する
- **THEN** その枠が選んだポケモンに差し替わり、HOME 使用率から性格・持ち物・特性・努力値・技が自動で入る

#### Scenario: 初回選出時の自動補完

- **WHEN** 技が未設定（空）のポケモンを初めて選出（選択）する
- **THEN** HOME 使用率から性格・持ち物・特性・努力値・技が自動で補完される

### Requirement: メガ進化（フォーム切替）

選択ポケモンの画像**または名前**をタップするとフォーム切替でき**なければならない（MUST）**。旧 champ-edge の
フォームボタンと同様に、`pokemon_data` にそのポケモンの**複数フォルム**（form 0〜19。通常の別フォルム
1-9 とメガ 10-19 を含む）がある場合、タップで昇順に循環し、最後のフォルムの次は先頭（通常）に戻る。
切り替え時は種族値・タイプ・特性・名前・画像をそのフォルムに差し替え、技・努力値・性格・持ち物・ランク等は
維持する。フォルムが1つだけのポケモンは変化しない。種族値ポップアップや外部サイトリンクは設けない。

#### Scenario: 名前タップでフォーム切替

- **WHEN** 複数フォルムを持つポケモンの名前をタップする
- **THEN** 次のフォルムに切り替わり、種族値・タイプを反映してダメージが再計算される

### Requirement: 技の威力・回数効果ボタン

システムは技の効果をボタンで切り替えられるようにし**なければならない（MUST）**。旧 champ-edge の
`waza.py`（multi_hit / add_power / power_hosei / self・opponent の buff・debuff / other_effect /
タイプ変更技）を移植し、対象の技には技名の横（HP バー右端）に効果ボタンを出す。技の種別に応じて
ボタンの表示と挙動を変える:

- **威力・回数**（連続技＝タネマシンガン等、威力増加＝アクロバット/じしん等、威力補正＝はたきおとす等）:
  「×n」表示。タップで候補値を循環し、`MoveState` の multiHit/addPower/powerHosei に渡す（既定値は原典準拠）。
- **ランク変化（自分/相手）**（つるぎのまい等＝自分↑、インファイト等＝自分↓、あまえる等＝相手↓、いばる＝相手↑）:
  「＋」/「−」表示。**タップするたびに 1 回使用分のランク変化を対象（自分＝攻撃側／相手＝防御側）の
  boosts に累積適用**し、**長押しで適用分をすべてリセット**する。
- **タイプ変更技**（みずびたし＝相手をみず、もえつきる／でんこうそうげき＝自分のほのお／でんきを失う）:
  タップで適用/解除し、タイプ相性に反映する。
- **other_effect 系トグル**:
  - きょけんとつげき＝「受×2」（威力 ×2 のトグルで近似）
  - じこあんじ＝相手の boosts を自分へコピー（解除で復元）
  - スキルスワップ＝攻撃側↔防御側の特性を入替（解除で戻す）
  - コートチェンジ＝壁・定数ダメージ・ステルスロックを攻守で入替（解除で戻す）
  - オーラぐるま＝技タイプを あく として計算（解除で戻す）

ボタンは効果種別で色・ラベルを出し分け、適用状態が分かること。効果の無い技はボタンを出さず行を揃える。
トグル系は長押しでも解除できる。**ポケモンの選択を切り替えたときは、切替前の active ポケモンの技由来の効果
（タイプ/特性/壁等）を解除し、さらに手動・技由来を問わずランク（boosts）をクリアする**。

#### Scenario: 連続技の回数を変える

- **WHEN** タネマシンガンの効果ボタン（×3）をタップする
- **THEN** 回数が次の候補（×4）に変わり、ダメージ範囲・割合・確定数が再計算される

#### Scenario: ランク技を複数回積む／長押しでリセット

- **WHEN** つるぎのまいの効果ボタン（＋）を 2 回タップし、その後ボタンを長押しする
- **THEN** 2 回タップで攻撃ランクが +4 され、長押しで適用分がすべて解除されて元に戻る

#### Scenario: ポケモン切替でランク・技効果がリセットされる

- **WHEN** ランクや技効果を適用した状態で、パーティの別ポケモンを選択して active を切り替える
- **THEN** 切替前ポケモンのランク（手動含む）と技由来の効果が解除され、状態がリセットされる
