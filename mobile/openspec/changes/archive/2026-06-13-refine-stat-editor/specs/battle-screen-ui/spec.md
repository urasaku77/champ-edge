# 努力値ボタン・ランククリアの明確化

## MODIFIED Requirements

### Requirement: ステータス（努力値・ランク）の編集

システムは努力値とランクを編集できるようにし**なければならない（MUST）**。努力値は各 0〜32
スケール（champ-edge 準拠）で、**0／32 のワンクリックボタンと ±1 ステップボタン**で調整する（数値は
中央に表示）。ランクは各 −6〜+6（−／＋ステッパ）。実数値の
表示部をタップするとポップアップが開き（編集アイコンは置かない）、能力名・実数値・努力値・ランクを
整列して 1 画面に収めスクロール無しで表示すること。**ランクのクリアは「ランククリア」ラベル＋掃き出し
アイコンのボタン**で行い、全能力のランクを 0 に戻すと分かるようにする。変更すると実数値とダメージが
即時再計算されること。レベルは 50 固定、個体値は内部 31 固定で UI には出さない。

#### Scenario: 努力値変更で実数値が変わる

- **WHEN** ある能力の「32」ボタン（または＋ボタン）を押す
- **THEN** その能力の努力値が 32（または +1）になり、実数値とダメージが即時再計算される

#### Scenario: ランククリアでランクが 0 に戻る

- **WHEN** 「ランククリア」ボタンを押す
- **THEN** 全能力のランクが 0 に戻り、実数値とダメージが再計算される
