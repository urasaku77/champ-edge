# 対戦分析

## ADDED Requirements

### Requirement: 対戦分析

システムは対戦記録を集計して相手傾向を分析できるようにし**なければならない（MUST）**（旧 champ-edge の
対戦分析画面）。ドロワーメニュー「対戦分析」から全画面で開き、**期間・自分パーティ番号/連番**で絞り込み、
**対戦数と勝率**を表示する。相手ポケモンごとに、**表示モード3択**で次を集計表示する:

- **KPと勝率**：出現数（KP）と出現時勝率。
- **選出と勝率**：選出率（選出数÷出現数）と選出時勝率。
- **初手と勝率**：初手選出率（初手数÷出現数）と初手時勝率。

集計は相手パーティ6枠・相手選出4枠（初手は1枠目）を対戦単位で数える。**メガ統合**を ON にすると
メガ進化フォーム（form 10-19）を通常フォームに集約して数える。

操作パネルで**ソート（件数/勝率）・降順/昇順・値の表示形式（%／分数／両方）・メガ統合・表示件数（50/100）**を
切り替えられる。集計ロジックは UI 非依存に分離する。

#### Scenario: 表示モードを切り替える

- **WHEN** 分析画面で表示モードを「初手と勝率」に切り替える
- **THEN** 各相手ポケモンの初手選出率と初手時勝率が表示され、選んだソート/形式で並ぶ

#### Scenario: メガ統合

- **WHEN** メガ統合を ON にする
- **THEN** メガ前後（例：リザードンとメガリザードン）が同一ポケモンとして合算集計される

#### Scenario: 勝率でソートする

- **WHEN** ソートを「勝率」・降順にする
- **THEN** 相手ポケモンが勝率の高い順に並ぶ
