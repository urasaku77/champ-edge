# タイマー・カウンタ

## ADDED Requirements

### Requirement: タイマー・カウンタ

システムは対戦用のタイマーとカウンタを備え**なければならない（MUST）**（旧 champ-edge の
TimerFrame / CountersFrame 相当）。

- **タイマー**：中央列のタイマーアイコンでポップアップを開く。20 分（1200 秒）からのカウントダウンで、
  スタート/ストップの切替とリセットができる。残り時間表示（mm:ss）の背景色は原典と同一の閾値で変化する
  （残り 10 分以上＝緑／10 分未満＝青／5 分未満＝黄／3 分未満＝赤）。ポップアップを閉じても計測は継続し、
  作動中（または残り時間が初期値以外）は中央列のアイコンが残り時間（mm:ss、閾値色）表示になる。
- **カウンタ**：中央列のカウンターアイコンでポップアップを開く。カウンタを 2 個並べ、各カウンタに
  タイトル（自由入力）・カウント値（0〜99）・「−」「0」「＋」ボタンを備える。ポップアップを閉じても
  値とタイトルは保持される（PP 管理等に利用）。

#### Scenario: タイマーの開始と色変化

- **WHEN** タイマーをスタートし、残り時間が 3 分未満になる
- **THEN** カウントダウンが進行し、表示の背景が赤になる（閉じても計測継続・アイコンに残り時間表示）

#### Scenario: カウンタで PP を管理する

- **WHEN** カウンタのタイトルに技名を入れ「＋」を3回押してポップアップを閉じ、再度開く
- **THEN** カウントが 3 のまま保持されている

## MODIFIED Requirements

### Requirement: メニュー（隅のボタンから展開）

システムは中央列に操作アイコンとメニューを備え**なければならない（MUST）**。中央列は上から
メニュー（ハンバーガー）／HOME 使用率／類似パーティ検索／対戦記録のアイコン、天候・フィールドの
ドロップダウン、カウンター／タイマーのアイコンを縦に並べる（押しやすいよう間隔を空ける）。
ハンバーガーを押すとドロワーが開き、区切り線で「素早さ比較・重さ比較」「パーティ編集・ボックス編集」
「対戦履歴・対戦分析」「設定」の4グループを表示する。**素早さ比較・重さ比較・カウンター・タイマーは
実装済み**（各要件参照）。残りのアイコン/項目はプレースホルダ（押下で「今後実装予定」）で、
段階的に実装してよい。

#### Scenario: メニューを開く

- **WHEN** 中央列のハンバーガーメニューをタップする
- **THEN** 区切り線で分類された各メニュー項目（比較／パーティ／戦績／設定）が表示される
