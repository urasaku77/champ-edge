# メタモンのへんしん

## MODIFIED Requirements

### Requirement: 特性による天候・フィールドの自動反映

システムは場に出たポケモンの特性を自動反映し**なければならない（MUST）**。旧 champ-edge の after_appear と
同様に、ポケモンを選択（登場）した際、特性に応じて以下を反映する:

- **天候特性**（すなおこし→砂嵐／ひでり・ひひいろのこどう・メガソーラー→晴れ／あめふらし→雨／ゆきふらし→雪）
  および**フィールド特性**（エレキメイカー・ハドロンエンジン→エレキ／グラスメイカー→グラス／ミストメイカー→
  ミスト／サイコメイカー→サイコ）であれば、中央列の天候/フィールドを対応する値に変更する。
- **いかく**：相手の攻撃ランクを 1 段階下げる。場を離れる（別ポケモンへ切替）と元に戻す。
- **トレース**：相手の特性をコピーして自分の特性欄に反映する。切替で元の特性に戻す。
- **メタモン**：相手の active ポケモンを丸ごとコピーする（HP は メタモンの種族値 48 を維持し、A〜S・タイプ・
  特性・技を相手からコピー）。切替でメタモンの状態に復元する。

また**スキルリンク**を持つポケモンの連続技（タネマシンガン等）は、回数を最大（5）として計算する。

#### Scenario: 天候特性で天候が変わる

- **WHEN** すなおこしのポケモンを選択する
- **THEN** 天候が「砂嵐」に変わり、ダメージが再計算される

#### Scenario: いかくで相手の攻撃が下がる

- **WHEN** いかく持ちのポケモンを選択する
- **THEN** 相手の攻撃ランクが −1 され、相手の物理ダメージが下がる（別ポケモンへ切り替えると戻る）

#### Scenario: スキルリンクで連続技が最大回数になる

- **WHEN** スキルリンク持ちのポケモンでタネマシンガンを使う
- **THEN** 連続技の回数が 5 として計算される

#### Scenario: メタモンが相手をコピーする

- **WHEN** メタモンを選択する
- **THEN** 種族値（HP は48維持）・タイプ・特性・技が相手のものになり、別ポケモンへ切り替えると元のメタモンに戻る
