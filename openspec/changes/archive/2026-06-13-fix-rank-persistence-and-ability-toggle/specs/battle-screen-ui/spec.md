# ランク変化の永続化・登場時ランク特性の拡張/無効化・天候フィールドのクリア

## MODIFIED Requirements

### Requirement: 技の威力・回数効果ボタン

システムは技の効果をボタンで切り替えられるようにし**なければならない（MUST）**。旧 champ-edge の
`waza.py`（multi_hit / add_power / power_hosei / self・opponent の buff・debuff / other_effect /
タイプ変更技）を移植し、対象の技には技名の横（HP バー右端）に効果ボタンを出す。技の種別に応じて
ボタンの表示と挙動を変える:

- **威力・回数**（連続技＝タネマシンガン等、威力増加＝アクロバット/じしん等、威力補正＝はたきおとす等）:
  「×n」表示。タップで候補値を循環し、`MoveState` の multiHit/addPower/powerHosei に渡す（既定値は原典準拠）。
- **ランク変化（自分/相手）**（つるぎのまい等＝自分↑、インファイト等＝自分↓、あまえる等＝相手↓、いばる＝相手↑）:
  「＋」/「−」表示。**タップするたびに 1 回使用分のランク変化を対象（自分＝攻撃側／相手＝防御側）の
  boosts に累積適用**し、**長押しで適用分をすべてリセット**する。
- **タイプ変更技**（みずびたし＝相手をみず、もえつきる／でんこうそうげき＝自分のほのお／でんきを失う）:
  タップで適用/解除し、タイプ相性に反映する。
- **other_effect 系トグル**:
  - きょけんとつげき＝「受×2」（威力 ×2 のトグルで近似）
  - じこあんじ＝相手の boosts を自分へコピー（解除で復元）
  - スキルスワップ＝攻撃側↔防御側の特性を入替（解除で戻す）
  - コートチェンジ＝壁・定数ダメージ・ステルスロックを攻守で入替（解除で戻す）
  - オーラぐるま＝技タイプを あく として計算（解除で戻す）

ボタンは効果種別で色・ラベルを出し分け、適用状態が分かること。効果の無い技はボタンを出さず行を揃える。
トグル系は長押しでも解除できる。**ポケモンの選択を切り替えたときは、切替前の active ポケモン自身の
ランク（boosts、手動・技由来を問わず）をクリアし、自分に紐づくトグル（タイプ変更/特性入替/壁等）を
解除する。ただしランク変化は「変化させられた側」に残るため、相手に入れたランク（あまえる等）は巻き戻さず、
その相手が場を離れたときにクリアされる**。

#### Scenario: 連続技の回数を変える

- **WHEN** タネマシンガンの効果ボタン（×3）をタップする
- **THEN** 回数が次の候補（×4）に変わり、ダメージ範囲・割合・確定数が再計算される

#### Scenario: ランク技を複数回積む／長押しでリセット

- **WHEN** つるぎのまいの効果ボタン（＋）を 2 回タップし、その後ボタンを長押しする
- **THEN** 2 回タップで攻撃ランクが +4 され、長押しで適用分がすべて解除されて元に戻る

#### Scenario: キングシールドで相手の攻撃が下がる

- **WHEN** キングシールドの効果ボタン（−）をタップする
- **THEN** 相手（攻撃側）の『こうげき』ランクが −1 され、相手の物理ダメージが下がる

#### Scenario: 自分が引っ込んでも相手に入れたランクは残る

- **WHEN** あまえるで相手の攻撃を下げた後、自分のパーティの別ポケモンへ切り替える
- **THEN** 自分のランクはクリアされるが、相手の攻撃ダウンは残り、相手が場を離れるとクリアされる

### Requirement: 天候・フィールド・壁の指定

システムは天候・フィールド・壁を指定できるようにし**なければならない（MUST）**。天候（なし/晴れ/雨/
砂嵐/雪）とフィールド（なし/エレキ/サイコ/グラス/ミスト）は**中央列**（メニュー/HOME アイコンを縦に
並べたその下のドロップダウン）から、壁（壁なし/物理壁＝リフレクター/特殊壁＝ひかりのかべ/両壁＝
オーロラベール）は選択ポケモンから指定でき、いずれも計算に反映されること。中央列の天候/フィールドの
ドロップダウンは**長押しで「なし」へクリア**できる。テラスタイプ変更は当面対象外（将来再導入候補）。

#### Scenario: 天候変更でダメージが変わる

- **WHEN** 中央列の天候を「雨」にする
- **THEN** みずタイプ技が強化され、ほのおタイプ技が弱化されてダメージが再計算される

#### Scenario: 天候/フィールドを長押しでクリアする

- **WHEN** 中央列の天候（またはフィールド）のドロップダウンを長押しする
- **THEN** その天候（フィールド）が「なし」になり、ダメージが再計算される

### Requirement: 特性による天候・フィールドの自動反映

システムは場に出たポケモンの特性を自動反映し**なければならない（MUST）**。旧 champ-edge の after_appear と
同様に、ポケモンを選択（登場）した際、特性に応じて以下を反映する:

- **天候特性**（すなおこし→砂嵐／ひでり・ひひいろのこどう・メガソーラー→晴れ／あめふらし→雨／ゆきふらし→雪）
  および**フィールド特性**（エレキメイカー・ハドロンエンジン→エレキ／グラスメイカー→グラス／ミストメイカー→
  ミスト／サイコメイカー→サイコ）であれば、中央列の天候/フィールドを対応する値に変更する。
- **登場時ランク特性**（自動適用）:
  - **いかく**：相手の『こうげき』ランクを 1 段階下げる。
  - **ふとうのけん**：自分の『こうげき』ランクを 1 段階上げる。
  - **ふくつのたて**：自分の『ぼうぎょ』ランクを 1 段階上げる。
  - **ダウンロード**：相手の ぼうぎょ実数値 < とくぼう なら自分の『こうげき』、そうでなければ『とくこう』を 1 段階上げる。

  これらのランクは適用した側が場を離れても巻き戻さない（変化させられた側が場を離れるまで残る）。
  **特性チップの長押し詳細から、ポケモン単位で登場時ランク特性の自動適用を無効化できる**（無効化はパーティに永続）。
- **トレース**：相手の特性をコピーして自分の特性欄に反映する。切替で元の特性に戻す。
- **メタモン**：相手の active ポケモンを丸ごとコピーする（HP は メタモンの種族値 48 を維持し、A〜S・タイプ・
  特性・技を相手からコピー）。切替でメタモンの状態に復元する。

また**スキルリンク**を持つポケモンの連続技（タネマシンガン等）は、回数を最大（5）として計算する。

#### Scenario: 天候特性で天候が変わる

- **WHEN** すなおこしのポケモンを選択する
- **THEN** 天候が「砂嵐」に変わり、ダメージが再計算される

#### Scenario: いかくで相手の攻撃が下がり、自分が引っ込んでも残る

- **WHEN** いかく持ちのポケモンを選択し、その後で自分の別ポケモンへ切り替える
- **THEN** 相手の攻撃ランクが −1 され、切り替えても相手の −1 は残る（相手が場を離れるとクリアされる）

#### Scenario: 登場時ランク特性を無効化する

- **WHEN** いかく持ちのポケモンの特性を長押しし、「登場時のランク効果を無効化」をオンにする
- **THEN** 以降そのポケモンを選出しても相手の攻撃ランクは下がらない

#### Scenario: スキルリンクで連続技が最大回数になる

- **WHEN** スキルリンク持ちのポケモンでタネマシンガンを使う
- **THEN** 連続技の回数が 5 として計算される

#### Scenario: メタモンが相手をコピーする

- **WHEN** メタモンを選択する
- **THEN** 種族値（HP は48維持）・タイプ・特性・技が相手のものになり、別ポケモンへ切り替えると元のメタモンに戻る
