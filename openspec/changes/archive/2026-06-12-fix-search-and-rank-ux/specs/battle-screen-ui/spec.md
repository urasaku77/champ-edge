# 検索照合とランク技操作の修正

## MODIFIED Requirements

### Requirement: 技の文字入力検索による選択

技は文字入力で検索して選択できる仕組みを提供し**なければならない（MUST）**。技名の一部を
入力すると候補が絞り込まれ、選んだ技がその枠に差し替わり、ダメージが再計算されること。
技候補は `pokemon.db` の `waza_data` から取得する（DB 未接続時はサンプル技プールで代替）。
`waza_data` の技名は**ひらがな**格納のため、検索は**生入力**と**カタカナ→ひらがな変換**の両方で
部分一致させる（例: 「みずびたし」「ミズビタシ」どちらでもヒットする）。

#### Scenario: 技名検索で差し替え

- **WHEN** 技枠をタップし、検索欄に技名の一部を入力する
- **THEN** 一致する技候補が表示され、選択した技がその枠に設定されダメージが再計算される

#### Scenario: ひらがな技の検索

- **WHEN** 検索欄に「みずびたし」と入力する
- **THEN** ひらがな格納の「みずびたし」が候補に表示される

### Requirement: 特性・持ち物の編集

選択ポケモンの特性・持ち物をタップで変更できなければならない（MUST）。特性はそのポケモンの
候補（DB の ability1/2/3、未接続時はサンプル）から、持ち物は**文字入力検索**（DB の item_data）で
選ぶ。item_data は表記が混在（カタカナ/ひらがな）のため、検索は**生入力・カタカナ・ひらがなのいずれかで
部分一致**させる（いのちのたま／こだわりスカーフ等いずれも引ける）。**特性・持ち物の選択時は、その相手
ポケモンの HOME 使用率を各候補の横に表示し、使用率の高い順に並べ替える**。変更するとダメージが
再計算されること。チップは選択値のみ表示し、持ち物が「なし」のときは「もちものなし」と表示する。

#### Scenario: 特性の変更

- **WHEN** 特性をタップして候補から別の特性を選ぶ
- **THEN** 選択ポケモンの特性が更新され、特性補正を含めてダメージが再計算される

#### Scenario: 持ち物の検索（混在表記）

- **WHEN** 持ち物検索で「いのち」と入力する
- **THEN** ひらがな表記の「いのちのたま」が候補に表示される

### Requirement: 技の威力・回数効果ボタン

システムは技の効果をボタンで切り替えられるようにし**なければならない（MUST）**。旧 champ-edge の
`waza.py`（multi_hit / add_power / power_hosei / self・opponent の buff・debuff / other_effect /
タイプ変更技）を移植し、対象の技には技名の横（HP バー右端）に効果ボタンを出す。技の種別に応じて
ボタンの表示と挙動を変える:

- **威力・回数**（連続技＝タネマシンガン等、威力増加＝アクロバット/じしん等、威力補正＝はたきおとす等）:
  「×n」表示。タップで候補値を循環し、`MoveState` の multiHit/addPower/powerHosei に渡す（既定値は原典準拠）。
- **ランク変化（自分/相手）**（つるぎのまい等＝自分↑、インファイト等＝自分↓、あまえる等＝相手↓、いばる＝相手↑）:
  「＋」/「−」表示。**タップするたびに 1 回使用分のランク変化を対象（自分＝攻撃側／相手＝防御側）の
  boosts に累積適用**し、**長押しで適用分をすべてリセット**する。
- **タイプ変更技**（みずびたし＝相手をみず、もえつきる／でんこうそうげき＝自分のほのお／でんきを失う）:
  タップで適用/解除し、タイプ相性に反映する。
- **other_effect 系トグル**:
  - きょけんとつげき＝「受×2」（威力 ×2 のトグルで近似）
  - じこあんじ＝相手の boosts を自分へコピー（解除で復元）
  - スキルスワップ＝攻撃側↔防御側の特性を入替（解除で戻す）
  - コートチェンジ＝壁・定数ダメージ・ステルスロックを攻守で入替（解除で戻す）
  - オーラぐるま＝技タイプを あく として計算（解除で戻す）

ボタンは効果種別で色・ラベルを出し分け、適用状態が分かること。効果の無い技はボタンを出さず行を揃える。
トグル系は長押しでも解除できる。**ポケモンの選択を切り替えたときは、切替前の active ポケモンに対して
適用されていた技由来の効果（ランクは累積回数分／タイプ/特性/壁等）をすべて解除し、状態をリセットする**
（手動設定とは独立に技由来のみ戻す）。

#### Scenario: 連続技の回数を変える

- **WHEN** タネマシンガンの効果ボタン（×3）をタップする
- **THEN** 回数が次の候補（×4）に変わり、ダメージ範囲・割合・確定数が再計算される

#### Scenario: ランク技を複数回積む／長押しでリセット

- **WHEN** つるぎのまいの効果ボタン（＋）を 2 回タップし、その後ボタンを長押しする
- **THEN** 2 回タップで攻撃ランクが +4 され、長押しで適用分がすべて解除されて元に戻る

#### Scenario: 特性入替（スキルスワップ）

- **WHEN** スキルスワップの効果ボタンを ON にする
- **THEN** 攻撃側と防御側の特性が入れ替わり、特性補正を反映してダメージが再計算される（再タップで戻る）

#### Scenario: ポケモン切替で技効果がリセットされる

- **WHEN** 技効果を適用した状態で、パーティの別ポケモンを選択して active を切り替える
- **THEN** 切替前ポケモンの技由来の効果（ランク累積分含む）が解除され、状態がリセットされる
