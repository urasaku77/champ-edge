## Why

対戦画面の技効果ボタンは現在「威力・回数」（連続技回数・威力増加・威力補正＝`waza.py` の
multi_hit/add_power/power_hosei）のみ対応している。旧 champ-edge では同じ「技の右ボタン」で
**ランク上昇/下降技**（つるぎのまい等）・**相手ランク変化技**（あまえる等）・**特殊トグル技**
（きょけんとつげき「受×2」、オーラぐるま「はらぺこ/まんぷく」、じこあんじ/スキルスワップ/
コートチェンジ）・**タイプ変更技**（みずびたし/もえつきる/でんこうそうげき）も操作できた。
これらが未移植のため、対戦中にこれらの技を選んでも 1 回使用分のランク変化や効果を反映できない。
ダメージエンジン自体はランク（boosts）・タイプ・各種補正を計算できるので、UI と効果テーブルの
移植のみで対応できる。P1（コアダメージ計算）の完成度を上げる。

## What Changes

- `waza.py` の `self_buff_values` / `self_debuff_values` / `opponent_buff_values` /
  `opponent_debuff_values` / `other_effect_values` を Dart の効果テーブルへ移植する。
- 技効果ボタンを拡張し、技の種別に応じて挙動を変える:
  - **ランク変化技（自分/相手）**: ボタンで「1 回使用分」のランク変化を該当ポケモンの boosts に
    適用/解除（トグル）。ラベルは "+"/"−"（旧アプリ準拠）。
  - **特殊トグル技**: きょけんとつげき＝「受×2」トグル（有効時、相手が受けるダメージ 2 倍）。
    オーラぐるま＝「はらぺこ/まんぷく」トグル（はらぺこ時あくタイプ扱い）。じこあんじ等＝有効/無効。
  - **タイプ変更技**: みずびたし（相手をみずに）・もえつきる/でんこうそうげき（自分のほのお/でんき
    タイプを失う）をボタンで適用/解除。
- ボタンの色/ラベルは効果種別で出し分け、適用状態が分かるようにする。場に出し直す（ポケモン
  選択切替）と一過性の効果はリセットする（旧アプリの appear リセット準拠）。
- 既存の「威力・回数（×n）」効果ボタンの挙動は維持する。
