# 中央列の調整・カウンタ集約・空きスロット表示

## MODIFIED Requirements

### Requirement: メニュー（隅のボタンから展開）

システムは中央列に操作アイコンとメニューを備え**なければならない（MUST）**。中央列は上から
**メニュー（ハンバーガー）／タイマー**／HOME 使用率／類似パーティ検索／対戦記録のアイコン、
天候・フィールドのドロップダウン、カウンタの直接表示を縦に並べる。**アイコン（HOME/類似/対戦記録）は
間隔を詰め、天候・フィールド・タイマー・カウンタは間隔を広くとる**（押しやすさ優先。スクロール・
縦オーバーフローはしない）。ハンバーガーを押すとドロワーが開き、区切り線で「素早さ比較・重さ比較」
「パーティ編集・ボックス編集」「対戦履歴・対戦分析」「設定」の4グループを表示する。
**素早さ比較・重さ比較・カウンター・タイマーは実装済み**（各要件参照）。残りの項目はプレースホルダ
（押下で「今後実装予定」）で、段階的に実装してよい。

#### Scenario: メニューを開く

- **WHEN** 中央列のハンバーガーメニューをタップする
- **THEN** 区切り線で分類された各メニュー項目（比較／パーティ／戦績／設定）が表示される

### Requirement: タイマー・カウンタ

システムは対戦用のタイマーとカウンタを備え**なければならない（MUST）**（旧 champ-edge の
TimerFrame / CountersFrame 相当）。タイマー・カウンタは**ポップアップではなく中央列に直接表示**し、
表示はサイズ固定でレイアウトがずれないようにする（文字は小さくてよい）。中央列の幅は従来どおりで
ポケモン情報の幅を変えない。

- **タイマー**：中央列の**ハンバーガー直下**に残り時間（mm:ss）を常時表示する。20 分（1200 秒）からの
  カウントダウンで、**一回押し＝スタート/ストップ・長押し＝リセット**（ボタンは設けない）。表示の
  背景色は原典と同一の閾値で変化する（残り 10 分以上＝緑／10 分未満＝青／5 分未満＝黄／3 分未満＝赤）。
- **カウンタ**：中央列にカウンタ[0] を直接表示する。上段に値・下段に `−` `＋`（0〜99）を横並びで配置。
  **ダブルタップで 3 個のカウンタを扱う全体ポップアップ**（各タイトル入力・−/＋/0・全リセット）を開き、
  **長押しでカウンタ[0] を 0 リセット**する。カウンタ[0] はポップアップと共有。値とタイトルは
  **対戦記録または明示リセットまで永続化**する（PP 管理等に利用）。メニュー側にカウンタ項目は設けない。

#### Scenario: タイマーの開始と色変化

- **WHEN** 中央列のタイマーを一回押してスタートし、残り時間が 3 分未満になる
- **THEN** カウントダウンが進行し、表示の背景が赤になる（長押しでリセットできる）

#### Scenario: カウンタのポップアップと永続化

- **WHEN** 中央列のカウンタをダブルタップしてポップアップを開き、タイトルと値を設定してアプリを再起動する
- **THEN** カウントとタイトルが保持されている（中央列のカウンタ[0] にも反映される）

### Requirement: パーティと選出（画像・番号表示・最大3体）

各側はパーティ 6 体を**画像**で横並び表示し**なければならない（MUST）**。選出は独立した欄を設けず、
パーティのポケモンに**選んだ順の番号（1〜3、最大 3 体）**を重ねて表示すること。パーティの
ポケモンをタップすると詳細表示の対象（選択ポケモン）になり、選出が 3 体未満なら選出に追加される。
選出が 3 体のときは通常タップでは選出を変えず、**長押しで選出から解除**する。
**ポケモンが未設定の枠（空きスロット）は、アイコンを出さず白背景＋枠のみを表示する**
（選出番号も出さない）。

#### Scenario: タップで選出と詳細切替

- **WHEN** 選出が 3 体未満の状態でパーティのポケモンをタップする
- **THEN** そのポケモンが選出に番号付きで追加され、選択ポケモンの基本情報・技がそのポケモンに更新される

#### Scenario: 3 体選出済みでのタップ

- **WHEN** 既に 3 体選出済みの状態で未選出のポケモンをタップする
- **THEN** 選出は変化せず、詳細表示のみそのポケモンに切り替わる

#### Scenario: 長押しで選出解除

- **WHEN** 選出中のポケモンをパーティで長押しする
- **THEN** そのポケモンが選出から外れる

#### Scenario: 空きスロットの表示

- **WHEN** パーティに未設定の枠がある
- **THEN** その枠はアイコンを出さず、白背景に枠線のみで表示される（例外やプレースホルダ文字を出さない）
