# 特性・持ち物の効果詳細ポップアップ

## MODIFIED Requirements

### Requirement: 特性・持ち物の編集

選択ポケモンの特性・持ち物をタップで変更できなければならない（MUST）。特性はそのポケモンの
候補（DB の ability1/2/3、未接続時はサンプル）から、持ち物は**文字入力検索**（DB の item_data）で
選ぶ。item_data は表記が混在（カタカナ/ひらがな）のため、検索は**生入力・カタカナ・ひらがなのいずれかで
部分一致**させる（いのちのたま／こだわりスカーフ等いずれも引ける）。**特性・持ち物の選択時は、その相手
ポケモンの HOME 使用率を各候補の横に表示し、使用率の高い順に並べ替える**。変更するとダメージが
再計算されること。チップは選択値のみ表示し、持ち物が「なし」のときは「もちものなし」と表示する。
さらに、**特性・持ち物のチップを長押しすると、その効果詳細（`ability_data.effect` / `item_data.effect`）を
ダイアログ表示する**（旧 champ-edge の右クリック効果詳細に相当）。

#### Scenario: 特性の変更

- **WHEN** 特性をタップして候補から別の特性を選ぶ
- **THEN** 選択ポケモンの特性が更新され、特性補正を含めてダメージが再計算される

#### Scenario: 持ち物の検索（混在表記）

- **WHEN** 持ち物検索で「いのち」と入力する
- **THEN** ひらがな表記の「いのちのたま」が候補に表示される

#### Scenario: 効果詳細の表示

- **WHEN** 特性チップ（または持ち物チップ）を長押しする
- **THEN** その特性/持ち物の効果説明がダイアログで表示される
